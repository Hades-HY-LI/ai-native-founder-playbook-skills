# Hiring

Use this reference to decide what to hire, automate, outsource, or defer.

## Bottleneck First

Do not start with roles. Start with the constraint:

- demand generation;
- sales conversion;
- onboarding and customer success;
- product quality;
- infrastructure and reliability;
- data and evals;
- founder time;
- operations.

## Decision Matrix

| Work Type | Default Move |
| --- | --- |
| Repetitive and low judgment | Automate or document |
| High judgment and recurring | Hire |
| Specialized and intermittent | Contract |
| Core to product advantage | Keep internal |
| Unvalidated | Founder owns until learned |
