# MVP Scope

Use this reference to cut an MVP to the smallest product that proves value.

## MVP Definition

The MVP is not the first complete product. It is the smallest reliable workflow that proves the customer can reach a valuable outcome.

## Scope Cut

Keep:

- one customer segment;
- one primary workflow;
- one success metric;
- minimum onboarding;
- manual operations where automation is not yet the risk;
- narrow integrations only if they are required for proof.

Defer:

- admin panels;
- broad customization;
- multi-person permissions;
- full analytics suites;
- complex billing;
- secondary personas;
- noncritical automations.
