# Founder-Led Sales

Use this reference for early sales before the motion is repeatable.

## Sales Loop

1. Select a narrow target account or user segment.
2. Research the workflow and trigger.
3. Send specific outreach.
4. Run discovery before demo.
5. Demo only the workflow tied to the pain.
6. Ask for a concrete next step.
7. Record objections and buying signals.
8. Update positioning, product, or ICP weekly.

## Discovery Questions

```text
What triggered interest?
How do you solve this today?
What happens when this goes wrong?
Who feels the cost?
What would make this a priority now?
How would you measure success?
What would block adoption?
```
