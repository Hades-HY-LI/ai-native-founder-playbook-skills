# Launch Operating System

Use this reference to create repeatable AI-assisted launch operations.

## Weekly Loops

Run these loops weekly:

- pipeline and outreach review;
- discovery notes synthesis;
- onboarding issue review;
- product feedback clustering;
- pricing and objection review;
- metric and decision review.

## Automation Candidates

AI can help with:

- account research;
- message drafts;
- call note summaries;
- objection clustering;
- FAQ and docs updates;
- onboarding checklists;
- weekly launch memo.

Keep human approval for:

- pricing changes;
- customer commitments;
- public claims;
- support escalation;
- product roadmap decisions.

## Output Format

```text
Recurring workflows:
AI-assisted steps:
Human approval gates:
Weekly dashboard:
Decision meeting agenda:
Artifacts to maintain:
```
