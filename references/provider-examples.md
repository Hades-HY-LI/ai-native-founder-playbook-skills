# Provider Examples

These examples are optional references. The skills do not require any provider.

## How To Use This Reference

Use provider names only when a founder asks which existing product might fill a generic AI role. Do not rewrite the core workflow around a vendor.

## Example Mapping

| Generic Role | Anthropic Examples | OpenAI Examples | Google Examples |
| --- | --- | --- | --- |
| Chat assistant | Claude | ChatGPT | Gemini |
| Coding agent | Claude Code | Codex | Gemini CLI |
| Collaborative workspace | Claude projects or workspace-style flows | ChatGPT projects or workspace-style flows | Gemini with Workspace-style flows |
| Model/API platform | Anthropic API | OpenAI API | Gemini API or AI Studio |
| Workflow agent | Agentic workflows built on Anthropic tools | Agent workflows built with OpenAI tools | Agent workflows built with Google tools |

Product names change. Verify current names and capabilities from official provider docs before publishing detailed setup instructions.
