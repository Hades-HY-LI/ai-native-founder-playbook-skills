# Stage Input Guide

Use this guide when a founder wants quick results from the skill pack.

## Two Ways To Start

Use guided intake when you do not have a complete brief:

```text
Use $founders-playbook-router to interview me, diagnose my stage, and recommend what to do next.
```

Use a structured brief when you already know the details:

## Universal Input Template

```text
Company/stage:
Target customer:
Problem hypothesis:
Current product status:
Existing users/customers:
Current AI tools:
Biggest constraint:
Desired output:
```
